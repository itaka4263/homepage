
```python
import psutil

# プロセスID（PID）を指定
pid = 1234  # 取得したいプロセスのPIDを設定してください

try:

    process = psutil.Process(pid)
```

このスクリプトは指定したプロセスのCPU使用率を取得し、それをコンソールに表示します。`pid`には取得したいプロセスのプロセスIDを設定してください。
